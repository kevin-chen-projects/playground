YouTube Data API v3 key

Slop Shield uses the YouTube Data API to read channel statistics. The free tier gives 10,000 quota units per day per Google Cloud project — more than enough for personal use. Create a key in the Google Cloud Console and enable the "YouTube Data API v3". Restrict it to HTTP referrers for safety.

Cache

Channel analyses are cached locally for 6 hours to reduce API quota usage.

Status of detection layers

  • Channel heuristics: active (upload cadence, channel age, view:sub ratio, duration consistency, title entropy, description boilerplate).
  • Voice / TTS: not yet implemented. Planned: AASIST-L ONNX in-browser.
  • Transcript text: not yet implemented. Planned: Fast-DetectGPT in-browser.
  • Face / avatar: not yet implemented. Planned: MediaPipe blink + head-pose heuristics.
